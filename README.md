# 📊 VST Limited Executive Sales Dashboard

A professional **Power BI Executive Dashboard** developed for **VST Limited** to monitor key business performance metrics, including revenue, profit, orders, customer segments, products, and regional sales. The dashboard enables interactive analysis and supports executive-level decision-making through dynamic visualizations and KPIs.

---

## 🚀 Features

* Executive KPI Cards (Revenue, Profit, Orders, Average Order Value)
* Revenue Analysis by Region and State
* Customer Segment Performance
* Product-wise Revenue Analysis
* Salesperson Performance
* Interactive Region and City Filters
* Business-focused Visualizations

---

## 🛠️ Tools & Technologies

* Microsoft Power BI
* Power Query
* DAX
* Data Modeling
* Interactive Dashboard Design

---

## 📈 Dashboard Insights

* Monitor overall business performance through key KPIs.
* Identify top-performing products and salespersons.
* Compare revenue across regions and states.
* Analyze customer segment contributions.
* Support strategic business decision-making.
